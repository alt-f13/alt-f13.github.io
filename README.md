Установка:
----------

```
git clone https://github.com/alt-f13/cm-spb.ru.git
cd cm-spb.ru
npm install
bower install
npm i -g docpad
```

Запуск
------

```
npm start

```

dev сервер будет доступен на [http://localhost:3000](http://localhost:3000)

Публикация
----------

```
npm gh-deploy
```
